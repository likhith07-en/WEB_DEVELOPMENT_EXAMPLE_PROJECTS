# 📝 To-Do List Web App
A simple To-Do List web application built with **HTML, CSS, and JavaScript**.  
It allows users to manage their daily tasks easily.
## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Edit or delete tasks
- Save tasks using browser localStorage (persistent even after refresh)
---
## 🛠 Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Storage**: LocalStorage (browser-based)
---
## 📂 Project Structure
📁 ToDo-List ┣ 📜 index.html ┣ 📜 style.css ┗ 📜 script.js
## ⚙️ Installation & Usage
1. Clone this repository or download the source code.
2. Open `index.html` in your browser.
3. Start adding and managing tasks.
